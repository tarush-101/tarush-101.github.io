Music Intelligence · Machine Learning · Full-Stack

Moodify

View on GitHub ↗

A full-stack music intelligence application that ingests a Spotify playlist and clusters tracks by mood using Gaussian Mixture Models with custom emotional feature weighting. Built with a two-phase hybrid LLM correction pipeline and a Streamlit analytics dashboard featuring t-SNE visualisation.

Python FastAPI Streamlit Gaussian Mixture Model Spotify API Gemini LLM Genius API t-SNE
  • Clusters tracks by mood using GMM with custom weighting across 9 audio dimensions — energy, valence, danceability, BPM, acousticness, and more
  • Mood Transition Journey: a 2D emotional coordinate system (Valence × Energy) using linear interpolation to generate sequenced playlists across emotional states
  • Two-phase LLM pipeline: Phase 1 uses Gemini for fast batch classification; Phase 2 fetches lyrics via Genius API for low-confidence tracks and re-classifies
  • Streamlit dashboard with t-SNE visualisation, mood distribution charts, AI confidence breakdowns, and LLM correction tracking
  • Personal contributions: research design, core ML logic, analytics dashboard, and coding
